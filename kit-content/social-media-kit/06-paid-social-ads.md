# 06 — Paid Social Ads

This file gives you copy-paste ad copy templates for Meta (Facebook/Instagram), TikTok, and LinkedIn, audience targeting strategies, A/B testing frameworks, budget allocation guides, and complete retargeting sequences that turn cold audiences into paying customers.

## SECTION 1: META ADS (FACEBOOK & INSTAGRAM)

### 1. Meta Ad Copy Generator (Complete Campaign)

**Platform:** Facebook, Instagram
**Best for:** Creating full ad campaigns with multiple copy variations

```
I'm running Meta ads (Facebook + Instagram) for [BUSINESS/BRAND].

Product/Service: [DESCRIBE IN DETAIL]
Price: [PRICE]
Target audience: [DESCRIBE — demographics, interests, pain points]
Campaign goal: [AWARENESS / TRAFFIC / LEADS / CONVERSIONS / SALES]
Landing page URL: [URL]
Offer (if any): [DISCOUNT, FREE TRIAL, FREE SHIPPING, etc.]

Create a complete ad campaign with 3 ad sets, each with 3 ad variations:

AD SET 1: PAIN POINT FOCUSED
Ad 1A — Short copy (under 125 characters for primary text):
- Primary text
- Headline (40 chars max)
- Description (25 chars max)
- CTA button choice

Ad 1B — Medium copy (125-250 characters):
- Primary text
- Headline
- Description
- CTA button choice

Ad 1C — Long copy (250-500 characters):
- Primary text (story-driven)
- Headline
- Description
- CTA button choice

AD SET 2: BENEFIT/TRANSFORMATION FOCUSED
Ad 2A — Short copy
Ad 2B — Medium copy
Ad 2C — Long copy
(Same format as above)

AD SET 3: SOCIAL PROOF FOCUSED
Ad 3A — Short copy (using testimonial/results)
Ad 3B — Medium copy (case study style)
Ad 3C — Long copy (before/after story)
(Same format as above)

For each ad, also provide:
- Visual direction (what the image or video should show)
- Recommended format (single image, carousel, video, collection)
- Estimated audience size guidance
- A/B testing priority (which element to test first)

My brand voice: [DESCRIBE]
My best-performing organic content: [DESCRIBE 2-3 TOP POSTS]
My competitors' ads (if known): [DESCRIBE]
```

**Pro Tips:**
- Start with 3 ad variations per ad set — let Meta's algorithm find the winner
- Test short copy first — it typically outperforms long copy for cold audiences
- Use your best-performing organic content as ad creative — it's already proven

### 2. Facebook Lead Generation Ad Copy

**Platform:** Facebook
**Best for:** Lead form ads that capture emails or inquiries

```
I want to run Facebook lead generation ads for [BUSINESS].

My lead magnet/offer: [DESCRIBE — free guide, consultation, demo, discount, etc.]
My target audience: [DESCRIBE]
My ultimate goal: [WHAT HAPPENS AFTER THEY SUBMIT THE FORM — email sequence, sales call, etc.]

Create 5 lead gen ad variations:

For each ad:
1. Primary text (the main ad copy)
2. Headline
3. Description
4. CTA button text
5. Lead form headline
6. Lead form description
7. Thank you screen text
8. Visual direction

Ad styles to include:
1. Direct value proposition ("Get your free [THING]")
2. Curiosity-driven ("Discover the [SECRET/METHOD] that [RESULT]")
3. Problem-agitate-solve ("Struggling with [PROBLEM]? Here's the fix")
4. Social proof ("Join [NUMBER] [PEOPLE] who already [ACTION]")
5. Urgency/scarcity ("Only [NUMBER] spots left for [OFFER]")

Also provide:
- Recommended form fields (name, email, phone, custom question)
- Keep form fields to minimum — explain why
- Follow-up sequence after form submission (first 3 emails or messages)
```

## SECTION 2: TIKTOK ADS

### 4. TikTok Ad Copy & Script Generator

**Platform:** TikTok
**Best for:** Creating ads that blend with organic TikTok content

```
I want to run TikTok ads for [PRODUCT/SERVICE].

Target audience: [DESCRIBE — age, interests, TikTok behavior]
Product price: [PRICE]
Campaign objective: [TRAFFIC / CONVERSIONS / APP INSTALLS]
Landing page: [URL]

Create 5 TikTok ad scripts:

SCRIPT 1: THE UGC-STYLE TESTIMONIAL (30 sec)
- Hook: Person talking to camera with a bold claim
- Body: Quick demo or explanation
- Social proof: "I've been using this for [TIME] and [RESULT]"
- CTA: "Link in bio" or "Click the link"
- Visual: Casual, iPhone-shot aesthetic

SCRIPT 2: THE TREND-BASED AD (15-30 sec)
- Use a trending format adapted for the product
- Make it feel like organic content, not an ad
- Hook: Match the trending format's opening
- Product placement: Natural, not forced
- CTA: Subtle, woven into the content

SCRIPT 3: THE PROBLEM-SOLUTION AD (30 sec)
- Hook (3 sec): "If you struggle with [PROBLEM], watch this"
- Problem (7 sec): Describe the frustration
- Solution (15 sec): Show the product solving it
- CTA (5 sec): Clear next step

SCRIPT 4: THE BEFORE/AFTER AD (20 sec)
- Before (7 sec): Show life without the product
- Transition: Quick cut or visual effect
- After (8 sec): Show life with the product
- CTA (5 sec): "Don't wait, link below"

SCRIPT 5: THE LISTICLE AD (30-45 sec)
- Hook: "[NUMBER] reasons why [AUDIENCE] loves [PRODUCT]"
- Rapid-fire reasons with text overlays
- Social proof weaved in
- CTA at the end

For each script:
- Full spoken text (word for word)
- Text overlay directions
- Camera angles and transitions
- Music/sound recommendation
- Caption + hashtags
- Thumbnail/cover frame description

TikTok ad rules:
- Must not look like an ad in the first 3 seconds
- Vertical 9:16 format
- Captions/subtitles required
- Energy must match organic TikTok content
```

## SECTION 4: A/B TESTING FOR ADS

### 7. Ad A/B Testing Framework

**Platform:** All platforms
**Best for:** Systematically testing ad elements to improve performance

```
I'm running paid social ads on [PLATFORM(S)] for [PRODUCT/SERVICE] with a monthly budget of [AMOUNT].

Create a comprehensive A/B testing plan:

1. TESTING HIERARCHY (what to test first → last):
   - Priority 1: [ELEMENT] — why this matters most
   - Priority 2: [ELEMENT]
   - Priority 3: [ELEMENT]
   - Priority 4: [ELEMENT]
   - Priority 5: [ELEMENT]

2. FOR EACH ELEMENT, provide a test plan:

   a. CREATIVE (image/video):
   - Test A: [DESCRIBE]
   - Test B: [DESCRIBE]
   - Minimum budget per variation: [AMOUNT]
   - Minimum runtime: [DAYS]
   - Success metric: [CTR / ROAS / CPC / CPA]
   - Statistical significance threshold

   b. HEADLINE:
   - Test A: [DESCRIBE approach]
   - Test B: [DESCRIBE approach]
   - Same framework as above

   c. PRIMARY TEXT (ad copy):
   - Test A: Short copy
   - Test B: Long copy
   - Same framework

   d. AUDIENCE:
   - Test A: Interest-based targeting
   - Test B: Lookalike audience
   - Test C: Broad targeting
   - Same framework

   e. PLACEMENT:
   - Test A: Automatic placements
   - Test B: Manual (feed only)
   - Same framework

3. TESTING CALENDAR (month-by-month plan):
   - Month 1: Test [WHAT] — expected learning
   - Month 2: Test [WHAT] — builds on Month 1
   - Month 3: Test [WHAT] — builds on Month 2

4. DECISION FRAMEWORK:
   - When to declare a winner (minimum data needed)
   - When to kill a losing variation
   - How to implement learnings into future campaigns
   - How to document and track results

5. BUDGET ALLOCATION FOR TESTING:
   - What % of budget goes to testing vs. scaling winners
   - Minimum budget per test
   - Maximum number of simultaneous tests

My current best-performing ad: [DESCRIBE]
My current CPA/ROAS: [NUMBER]
My target CPA/ROAS: [NUMBER]
```

4. Visual direction

- Recommended testing order (which to test first)
- Expected budget needed to test all 10 (with minimum viable spend)
