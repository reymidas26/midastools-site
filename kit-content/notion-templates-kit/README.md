## How to Use This Kit

### Step 1: Pick Your Category

Start with the file that matches your most urgent need. You do not need to set up everything at once.

### Step 2: Copy the Template Structure into Notion

Each template includes:
- **Database properties** (columns, types, formulas) -- set these up in your Notion database
- **Page structure** -- the layout and sections for each page/entry
- **AI prompts** -- copy-paste these into Notion AI or ChatGPT to generate content

### Step 3: Customize the AI Prompts

Every prompt in this kit is a starting point. Replace the bracketed placeholders `[like this]` with your specific details for better results.

### Step 4: Build Your System Over Time

Start with one template. Get it working. Then layer on the next one. A half-used system beats a perfectly designed one you never open.

## How the AI Prompts Work

Each template includes prompts formatted in code blocks. These prompts work with:

1. **Notion AI** -- Highlight text in Notion, press `Space`, or use `/ai` to invoke Notion AI. Paste the prompt.
2. **ChatGPT / Claude** -- Copy the prompt, paste it into your AI tool of choice, then paste the output back into Notion.
3. **Notion AI Autofill** -- Some prompts are designed to work with Notion's database AI autofill feature for automatic property generation.

### Prompt Format

Every prompt follows this pattern:

```
You are a [role]. [Context about the task].

Given the following information:
- [Input 1]
- [Input 2]

Generate [specific output format].
```

Replace bracketed items with your real data.

## Tips for Maximum Value

1. **Start small.** Pick 3-5 templates that solve your biggest pain points right now.
2. **Use linked databases.** Connect templates across categories -- link your CRM contacts to your content calendar, your project tasks to your OKRs.
3. **Automate with AI autofill.** Set up Notion AI autofill on database properties so new entries get AI-generated content automatically.
4. **Create template buttons.** Once you customize a template, save it as a Notion template button so you can stamp out new instances instantly.
5. **Review weekly.** The best system is one you actually use. Set a weekly review to clean up, archive, and plan.
