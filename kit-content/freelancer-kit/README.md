## Quick Start (5 Minutes)

### Step 1: Choose Your AI Tool

These prompts work with any major AI assistant:

- **ChatGPT** (chat.openai.com — any current model)
- **Claude** — claude.ai
- **Gemini** — gemini.google.com
- **Any local LLM** (LLaMA, Mistral, etc.)

### Step 2: Copy a Prompt

1. Open any file in this kit.
2. Find the prompt you need (each is clearly labeled).
3. Copy the entire prompt block.

### Step 3: Fill In Your Variables

Every prompt uses `[BRACKET VARIABLES]` like this:

```
[YOUR NAME] — Your full name or business name
[CLIENT NAME] — The client's name or company
[PROJECT TYPE] — What you're delivering (e.g., "website redesign")
```

Replace each bracket variable with your actual information before pasting into your AI tool.

### Step 4: Paste and Generate

Paste the completed prompt into your AI tool. Review the output, adjust as needed, and use it in your business.

---

## Tips for Best Results

### Be Specific with Variables
The more detail you put into the bracket variables, the better your output. Instead of `[PROJECT TYPE] = website`, try `[PROJECT TYPE] = responsive e-commerce website with Shopify integration`.

### Save Your Favorites
Once you find prompts you use repeatedly, save them in a text file or note with your variables pre-filled. This turns a 5-minute task into a 30-second task.

### Chain Prompts Together
Many of these prompts work best in sequence. For example:
1. Use the **Proposal Generator** to win the project.
2. Use the **Client Onboarding System** to kick off professionally.
3. Use **Weekly Client Updates** throughout the project.
4. Use the **Scope Protector** if the client asks for extras.
5. Use the **Invoice & Payment Chaser** to get paid on time.

### Customize the Tone
Add a line to any prompt to adjust the voice:
- `Write in a warm, conversational tone.`
- `Write in a formal, corporate tone.`
- `Match the tone of this previous email: [PASTE EMAIL]`

### Iterate
AI output is a first draft. Read it, tweak it, make it yours. The goal is to go from blank page to 80% done in seconds.
