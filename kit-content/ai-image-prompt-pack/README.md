## Tips for Best Results

### General Tips
1. **Be specific** — "a golden retriever puppy sleeping on a red velvet couch in soft afternoon window light" beats "a cute dog" every time
2. **Include lighting** — lighting makes or breaks an image. Specify "soft diffused light", "golden hour", "dramatic rim lighting", etc.
3. **Mention camera details** — "shot with a Canon EOS R5, 85mm lens, f/1.4, shallow depth of field" adds photorealism
4. **State what you DON'T want** — add "no text", "no watermarks", "no borders" to avoid common issues
5. **Specify aspect ratio** — always include your target dimensions to avoid awkward cropping

### Platform-Specific Tips

**ChatGPT (native image generation):**
- Naturally understands long, descriptive prompts
- Great with text rendering in images
- Specify "photorealistic" or "illustration" to set the style
- Supports 1024x1024, 1024x1792 (portrait), 1792x1024 (landscape)

**Midjourney:**
- Add `--ar 16:9` for aspect ratio; add `--v 7` (or the newest version flag shown in-app) to get the latest quality
- Use `--style raw` for less Midjourney "house style"
- Add `--q 2` for higher quality (uses more compute)
- Shorter, punchier prompts often work better than long descriptions

**Stable Diffusion:**
- Use negative prompts to exclude unwanted elements
- Specify the model/checkpoint you're using for best results
- CFG scale 7-12 works well for most prompts
- Add "masterpiece, best quality" for quality boost

**Leonardo AI:**
- Select the right model for your use case (PhotoReal for photos, DreamShaper for illustrations)
- Use the negative prompt field for exclusions
- Guidance scale 7-9 is the sweet spot

## Frequently Asked Questions

**Q: Can I use these images commercially?**
A: The prompts are yours to use however you want. Image usage rights depend on the AI tool you use — check each platform's terms. As of 2026, ChatGPT, Midjourney, and most tools grant commercial usage rights on paid plans.

**Q: Why does the same prompt give different results on different platforms?**
A: Each AI model interprets prompts differently. That's normal. Use the "Best for" label on each prompt to pick the optimal tool, or try the prompt on multiple platforms and pick your favorite.

**Q: Can I modify these prompts?**
A: Absolutely. These are starting points. Mix, match, combine, and tweak to your heart's content. The best results come from iterating.

**Q: The image has weird text/artifacts. What do I do?**
A: Add "no text, no watermarks, no artifacts, clean image" to the end of your prompt. For text-heavy designs, ChatGPT's native image generation (or Gemini's) tends to handle text rendering best.

**Q: What aspect ratio should I use?**
A: Each prompt includes recommended sizing. As a quick reference:
- Instagram Post: 1:1 (1080x1080) or 4:5 (1080x1350)
- Instagram Story / TikTok / Reel: 9:16 (1080x1920)
- YouTube Thumbnail: 16:9 (1280x720)
- LinkedIn Banner: 4:1 (1584x396)
- Twitter/X Header: 3:1 (1500x500)
- Website Hero: 16:9 (1920x1080) or wider

**Q: How do I get consistent brand images?**
A: Include your exact brand colors (hex codes), typography style, and visual keywords in every prompt. Create a "brand prompt prefix" you paste before each prompt.
