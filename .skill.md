# Connected Services

These are the real-world tools available to this agent.

## Browser (Playwright)

- **Type**: mcp
- **Command**: `npx`
- **Description**: Browse the web, post to social media, fill forms, extract data — uses persistent login sessions
- **Capabilities**: browser.navigate, browser.click, browser.type, browser.screenshot, browser.tab_management
- **Rate limit**: 200/hour, 2000/day

## Google Workspace

- **Type**: cli
- **Command**: `gws`
- **Description**: Gmail, Calendar, Drive, Sheets, Docs — full Google Workspace access via CLI
- **Capabilities**: email.send, email.read, calendar.create, calendar.list, drive.list, drive.upload, sheets.read, sheets.write, docs.create
- **Rate limit**: 60/hour, 500/day

### Usage Examples

```bash
# Send an email
gws gmail send --to user@example.com --subject "Subject" --body "Body"

# List recent emails
gws gmail list --max-results 10

# Create a calendar event
gws calendar create --summary "Meeting" --start "2026-03-30T10:00:00" --end "2026-03-30T11:00:00"

# List files in Drive
gws drive list --max-results 10

# Read a spreadsheet
gws sheets read --spreadsheet-id <id> --range "A1:D10"
```

## Web Research

- **Type**: mcp
- **Command**: `npx`
- **Description**: Search the web, scrape pages, extract structured data, research prospects and competitors
- **Capabilities**: web.search, web.scrape, web.extract, web.crawl, web.map, web.agent, browser.interact
- **Rate limit**: 60/hour, 500/day

### Usage Examples

Use `firecrawl_search` to find pages, `firecrawl_scrape` to extract content,
`firecrawl_extract` for structured data, and `firecrawl_agent` for autonomous research.

- **Search**: `firecrawl_search` with query — returns URLs + content
- **Scrape**: `firecrawl_scrape` with URL — returns clean markdown
- **Extract**: `firecrawl_extract` with URLs + prompt — returns structured JSON
- **Map**: `firecrawl_map` with URL — discovers all pages on a site
- **Agent**: `firecrawl_agent` with natural language query — autonomous multi-site research
- **Interact**: `firecrawl_interact` — click, type, navigate on web pages
